-------------------------------------------------
Win11 Atlas
-------------------------------------------------


Steps:
------------------------

1. Install all Windows updates

2. RUN "Enable Features.cmd" and then RESTART

3. RUN "AME Wizard Beta.exe"

4. Import "Atlas Playbook.apbx" file and run

5. RESTART after playbook has completed

6. (optional)
	i.  UNZIP "WinSetView - edit explorer grouping.7z"
	ii. RUN "WinSetView.exe" to customize windows explorer

7. Enjoy :)
